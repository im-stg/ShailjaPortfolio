Research & Development Intern (JAN 2019 - OCT 2019)

Keysight Technologies International India Pvt. Ltd.

Designed and developed a single page web application ICM for the employees of Keysight. It aims at real-time tracking and management of instruments and targets two types of Keysight Users – Standard and Admin.

Standard Users are authorized for following operations.

  • View/ Download/Print instruments details.
  • Email instruments list.
  • Copy details of specified instrument.
  • Paginator to adjust view of data table according to user’s requirements
  • Searchbar to highlight searched instruments details
  • Sort headers to sort fields according to user’s requirements
  • View Employee details to whom instrument has been issued.
  • Users can also give feedback for their experience with ICM.

Users authenticated as Admins are authorized for the following operations too.

  • Addition of new instrument details.
  • For large entries admin can import data using import feature just by selecting file
  • Updation/ Deletion of any instrument details
  • Auto Email alerts to notify the user to return instruments in time.

Technologies used:

  • MongoDB
  • NodeJS
  • Express Middleware
  • Angular7
  • Postman
  • CSS
